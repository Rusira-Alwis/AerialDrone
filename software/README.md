# Software Folder

Contains INAV configuration exports, ground station scripts, and analysis notebooks.

## Contents
- `msp_parser.py` — Parses telemetry from INAV via serial.
- `requirements.txt` — Python dependencies.
- `notebooks/` — Jupyter notebooks for flight log visualization.
- `configs/` — Saved INAV CLI dumps for version control.

## Setup
Run `pip install -r requirements.txt` before executing scripts.
